# 🧂 NaCl-Analysis-21x4: Dataset Documentation

เอกสารฉบับนี้อธิบายรายละเอียดของชุดข้อมูลที่ใช้ในการวิเคราะห์ความคลาดเคลื่อนของอุปกรณ์วัดค่าความเค็ม (**NaCl**) ทั้ง 4 เครื่อง ภายใต้สภาวะอุณหภูมิที่แตกต่างกัน เพื่อใช้เป็นฐานข้อมูลหลักในการทำ Data Science และ Hardware Validation

---

## 📌 ที่มาของโปรเจกต์ (Project Background)

ศึกษาความคลาดเคลื่อน (**Variation**) ของอุปกรณ์วัด 4 ตัวที่ใช้โครงสร้างโมเดลเดียวกัน แต่มีชิ้นส่วนฮาร์ดแวร์ (ชิป, โพรบ) ต่างกัน เพื่อวิเคราะห์ผลกระทบจาก:
1. **Hardware Factors**: ความแตกต่างรายเครื่อง (Individual Variation) แม้จะใช้ Software เดียวกัน
2. **Temperature Factors**: ผลกระทบจากอุณหภูมิที่สูงขึ้น (Thermal Effect) ต่อความแม่นยำในการวัดค่า EC

---

## 📊 การออกแบบการทดลอง (Experimental Design)

ใช้รูปแบบ **Full Factorial Design** เพื่อเก็บข้อมูลให้ครอบคลุมทุกมิติความสัมพันธ์ของตัวแปร

### 1. ปัจจัยอิสระ (Independent Variables)

| ปัจจัย | รายละเอียด |
| :--- | :--- |
| **NaCl Concentration** | 21 ระดับ (0.0 ถึง 2.0, เพิ่มทีละ 0.1) |
| **Hardware Models** | 4 เครื่อง (`model01`, `model02`, `model03`, `model04`) |
| **Temperature Scope** | ครอบคลุมช่วงอุณหภูมิประมาณ **22°C - 95°C** |
| **Replications** | **35 รอบ** ต่อหนึ่งระดับความเข้มข้น (จัดเก็บแบบต่อเนื่อง) |

> [!NOTE]
> **หมายเหตุเกี่ยวกับลำดับการวัด (Sampling Note):**
> เดิมกำหนด Target Temp ไว้ 7 ระดับ ($25, 30, 40, 50, 60, 70, 80^\circ C$) โดยตั้งใจวัดระดับละ 5 รอบ อย่างไรก็ตาม เนื่องจากสภาวะหน้างานจริงอุณหภูมิมีการเปลี่ยนแปลงตลอดเวลา (Dynamic Temperature) ไม่สามารถควบคุมให้คงที่เป๊ะได้ จึงปรับเปลี่ยนกลยุทธ์การบันทึกดังนี้:
> 1. บันทึกค่าอุณหภูมิจริงที่เกิดขึ้นลงใน `RAW_Temp` โดยตรง
> 2. รันลำดับการวัด (Index) ต่อเนื่องเป็น **1-35** แทนการแยกกลุ่มอุณหภูมิ
> 3. วิธีนี้ช่วยให้ได้ข้อมูลที่เป็นแบบจำลองความจริง (Real-world Scenario) และเห็นการตอบสนองของเซนเซอร์ต่อการเปลี่ยนแปลงอุณหภูมิที่ต่อเนื่อง

---

### 2. รายละเอียดตัวแปรที่จัดเก็บ (Features)

* **`RAW_Temp`**: ค่าอุณหภูมิจริงขณะวัดจากThermometer (องศาเซลเซียส) - *ใช้เป็นตัวแปรอ้างอิงในสถานการณ์จริง เพื่อการวิเคราะห์*
* **`M_Temp`**: ค่าอุณหภูมิที่ mode0x แต่ละเครื่องประมวลผลได้
* **`EC`**: ค่าการนำไฟฟ้า (Electrical Conductivity) ที่วัดได้ ณ ขณะนั้น ของเครื่องแต่ละ model0x
* **`Rep`**: ลำดับการวัดที่ 1-35 (ครอบคลุมความผันแปรของอุณหภูมิทั้ง 7 ช่วงเป้าหมาย)
* **`Models`** | 4 เครื่อง (`model01`, `model02`, `model03`, `model04`) |
